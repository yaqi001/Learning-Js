# 初学 Js 需要注意的地方

1. alert() 函数在 JavaScript 中并不常用，但它对于代码测试非常方便。
   
   ~~~ html
   <!DOCTYPE html>
   <html>
	<body>

	<button type="button" onclick="alert('halo world')">say halo</button>

	<button type="button" onclick="myFunction()">点击这里</button>

	</body>
	</html>
   ~~~
  
   > **注意**
   > alter() 里的参数必须用单引号括起来！
